# Basic AI Medical Chatbot with RAG

This is an **AI-powered medical chatbot** built using **Streamlit**, **LangChain**, and **Groq LLM**, with **Retrieval-Augmented Generation (RAG)** on uploaded PDFs.

#Features
- Upload **medical PDFs**.
- Creates **FAISS-based knowledge base**.
- Ask questions and get **context-aware answers**.
- Uses **Hugging Face embeddings** and **Groq LLM**.



#Tech Stack
Frontend:Streamlit
Backend: LangChain, FAISS
LLM: Groq (Mixtral-8x7b-32768)
Embeddings: Hugging Face (`sentence-transformers/all-MiniLM-L6-v2`)
