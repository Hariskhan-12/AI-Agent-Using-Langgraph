# LangGraph AI Agent

This project demonstrates an AI Agent built with **LangGraph** and **Groq LLM**, supporting modular tools like web search and calculator.

## Features
- Modular structure (LLM, prompts, tools, agent)
- Groq-powered reasoning
- Extendable with new tools
- Streamlit interface

## Setup
1. Clone repo
2. Create `.env` file:
   GROQ_API_KEY=your_api_key_here
3. Install dependencies:
   pip install -r requirements.txt
4. Run app:
   streamlit run app.py
